# weather-information-system


A Python-based weather application that fetches real-time weather information using the WeatherAPI.

## Features

* Search weather by city name
* Display current temperature in Celsius and Fahrenheit
* Show humidity percentage
* Display weather condition
* Show local date and time
* Handle invalid city names gracefully

## Technologies Used

* Python
* Requests Library
* JSON
* WeatherAPI

## How It Works

1. User enters a city name.
2. The application sends a request to WeatherAPI.
3. Weather data is received in JSON format.
4. The program extracts relevant information and displays it in a user-friendly format.
